# Final Project: GreenTech Solutions
A responsive multi-page website promoting eco-friendly technology solutions.

**Project Description:**

GreenTech Solutions is a fictional company focused on sustainable, eco‑friendly technology.
This project demonstrates core front‑end development skills including semantic HTML structure,
responsive CSS layout, Flexbox, grid usage, JavaScript form validation, multi-page site navigation, and accessibility considerations.

**This site includes five pages:**

Home
About
Products
Blog
Contact

**Live Site:**

URL: https://nhstudent650.github.io/nhstudent650-web-project/final/index.html

**FEATURES**

HTML:

Semantic tags, organized multi-page structure, accessible labels, and alt text.

CSS:

Fully responsive layout, Flexbox navigation, grid based product layout, mobile optimized hero section, and custom color theme.

JavaScript:

Contact form validation, email format checking, error message, succcess message, and prevents submission unitl fields are valid.

**HOW TO RUN THE PROJECT**

Open the live URL listed above and browse the site normally. You could also download or clone the repository then open index.html in any browser and navigate using the top menu.

**TECHNOLOGIES USED**

HTML5, CSS3, JavaScript, GitHub, and Visual Studio Code.
