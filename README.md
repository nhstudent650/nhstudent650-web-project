# nhstudent650-web-project

Project 2 – HTML Foundation Page
    
Webpage Theme:
A basic webpage about PC building, explaining key components and including an image, list, table, and navigation links.

What I Learned:
How to structure a page using only HTML, apply inline styles, add accessible elements, and validate code with the W3C validator.

Challenges Faced:
Styling the page without CSS and handling the validator warning about the table border, which appears because the project uses HTML only.
